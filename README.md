# Craving Crusher 🤜

Your personal quit-smoking coach app. Track cravings, get AI coaching, breathing exercises, and see your progress.

## Deploy to Netlify (easiest — 2 minutes)

1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Create a free account if you don't have one
3. Drag and drop the **entire `craving-crusher-app` folder** onto the page
4. Done! You'll get a URL like `https://random-name.netlify.app`
5. Open that URL on your phone and add it to your home screen

### Add to Home Screen:
- **iPhone**: Open in Safari → Share button → "Add to Home Screen"
- **Android**: Open in Chrome → Three dots → "Add to Home Screen"

## AI Coach Setup (optional)

The app works without an API key (you'll get motivational messages instead of AI coaching).

To enable the AI coach:
1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Create an account and get an API key
3. Add some credits ($5 is plenty — each coaching message costs ~$0.003)
4. The app will ask for your key the first time you send a message

## Features

- 💬 **AI Coach** — Real-time craving support that talks you through it
- 🫁 **Breathing Exercise** — 4-4-6 technique to calm your nervous system
- 📊 **Stats** — Track streaks, success rate, money saved
- 📋 **Craving Log** — See patterns in when you crave most
- 📱 **Works offline** — Motivational messages work without internet
